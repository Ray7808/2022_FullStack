# SQL

SQL 就是 Structure Query Language，
中文是有架構的查詢語言。

- What is DBMS?

  DBMS 是 DataBase Management System(數據庫管理系統)。

  1. RBDMS

     R 是代表 Relational，
     RBDMS 代表關聯型資料庫。

  2. C.R.U.D

     Create, Read, Update, Delete，
     所有數據庫都涵蓋這四個功能。

  3. SQL and NoSQL

     SQL 是 Relational DataBase，
     因此任何不是 Relational DataBase 的就是 NoSQL。
